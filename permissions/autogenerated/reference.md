## Default Permission

Default permissions for the rust-network-runtime plugin

#### This default permission set includes the following:

- `allow-abort-http-request`
- `allow-eventsource-close`
- `allow-eventsource-connect`
- `allow-http-request`
- `allow-send-beacon`
- `allow-websocket-close`
- `allow-websocket-connect`
- `allow-websocket-send`
